Install Laravel5.0 via Composer

Posted on 2015-07-05

通过Composer快速(大雾)安装Laravel5.0

Mark下安装过程和其中的坑

Composer安装

下载Composer
# curl -sS https://getcomposer.org/installer | php

移动到全局环境变量里，就可以在任意地方执行composer
# mv composer.phar /usr/local/bin/composer

更多信息参考composer在线中文手册

PHP支持模块安装

下载模块
# yum install -y php56w-mysql,php56w-mysqlnd,php56w-xml,php56w-mbstring

重启apache
# service httpd restart

之前没有装这些模块，直接装Laravel了，结果每次都报错找不到函数！
安装Laravel之前一定要检查模块是否安装齐全

Laravel安装

进入apache文档目录
# cd /var/www/

安装Laravel5.0
# composer create-project laravel/laravel --prefer-dist

修改apache默认文档路径
# vi /etc/httpd/conf/httpd.conf
找到DocumentRoot字段 改为"/var/www/laravel/public/"

重启apache
# service httpd restart

修改storage权限
# chmod -R 777 storage
不改就莫名其妙的500！

访问试试

http://114.215.121.52
成功看到Laravel5！

换源

国内vps的话可以考虑换个composer源，让更新依赖包更快(没试过)
换源教程参考composer国内镜像
