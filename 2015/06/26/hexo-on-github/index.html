hexo-on-github

發表於 2015-06-26

最近在学习web，知识点太多，想写blog记录整理学习过程，github作为业内男性交友网站，提供了免费的github pages造福广大同胞。那么问题来了：静态blog框架哪家强。
hexo在github上搭建发布过程

1、node.js官网下载windows安装包，一键安装

2、git同样，一键安装，配置用户名、邮箱，连接github

3、打开git bash，进入文件夹，>npm install hexo -g

4、>hexo init xxx 后缀填写指定文件夹

5、下载主题，我使用的是hexo-theme-next github可下载

6、修改_config.yml中的theme

配置deploy

7、>hexo generate 静态化

8、>hexo deploy 没反应，检查了
